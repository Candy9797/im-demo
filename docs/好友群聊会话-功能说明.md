# 好友/群聊会话 - 功能说明

本文说明 `/chat` 路由下的好友与群聊会话功能：侧边栏、多会话切换、消息列表、引用/反应/编辑/撤回、草稿、滚动位置保持、免打扰、交易卡片等。当前为 **Mock 数据**，无后端连接。

---

## 一、功能概述

- **路由**：`/chat`。
- **布局**：左侧侧边栏（好友列表 + 群组列表），右侧主区域（当前会话标题、消息列表、输入区）。
- **会话类型**：单聊（C2C，按好友）、群聊（Group，按群）；会话 key：单聊 `c2c:userId`，群聊 `groupId`。
- **数据**：好友/群组来自 `MOCK_FRIENDS`、`MOCK_GROUPS`，消息来自 `chatSessionMock` 的 `getMessagesForConversation`；全部前端 Mock，无 WebSocket。

---

## 二、侧边栏与会话切换

| 组件 | 说明 |
|------|------|
| `ChatSessionSidebar` | 顶部返回、标题、Tab（好友/群组）、列表项（头像、名称、预览、未读数） |
| 选中态 | `activeConversation` 与列表项 id 一致时高亮 |
| 切换 | 点击好友或群组 → `selectFriend(id)` / `selectGroup(id)` → 更新 `activeConversation` 并写入 sessionStorage，便于刷新后恢复当前会话 |

当前会话会持久化到 sessionStorage，与「未发送草稿恢复」配合：刷新后先恢复当前会话，再按会话 key 从 IndexedDB 恢复草稿。

---

## 三、消息列表（主区域）

| 能力 | 实现 |
|------|------|
| 列表渲染 | `ChatSessionMain` 内使用 `react-virtuoso`，data 为当前会话排序后的 messages，computeItemKey 为 message.id |
| 单条渲染 | `MessageItem`：文本/图片/视频/文件/表情/交易卡片/引用/反应/编辑/撤回 |
| 滚动保持 | 切换会话时按 conversationKey 存「顶部可见 index」，切回时 scrollToIndex 恢复（见 [聊天会话滚动位置保持-技术说明.md](./聊天会话滚动位置保持-技术说明.md)） |
| 会话 ID 展示 | 标题栏展示 conversationKey，便于调试 |

消息按 `seqId ?? timestamp` 排序，来自 store 的 `messagesByConv[conversationKey]`。

---

## 四、输入区（ChatSessionInput）

- **文本**：Enter 发送、Shift+Enter 换行；支持引用回复（ChatSessionQuotePreview）。
- **表情/表情包**：EmojiPicker、StickerPicker，createPortal 挂到 body 定位。
- **图片/视频**：选择文件后校验类型与大小，sendImage/sendVideo，content 为 blob URL。
- **按住说话**：HoldToTalkButton，识别结果追加到输入框或可发送（见 [按住说话-语音输入技术说明.md](./按住说话-语音输入技术说明.md)）。
- **交易卡片**：工具栏「分享交易卡片」→ TradeCardShareModal，一键生成示例后分享到当前会话（见 [交易卡片分享-功能说明.md](./交易卡片分享-功能说明.md)）。
- **草稿**：当前会话 key 为 `chat:${conversationId}`，防抖写入 IndexedDB，切换会话或刷新后恢复（见 [未发送内容草稿存储-技术方案.md](./未发送内容草稿存储-技术方案.md)）。

---

## 五、对方正在输入

- **单聊**：`typingByUser[对方id]` 为 true 时，主区域底部显示「{对方昵称} 正在输入...」。
- **群聊**：`typingByGroup[groupId]` 为当前正在输入的 userId 列表，展示「A、B 正在输入...」或「N 人正在输入...」。
- **Mock 触发**：本方在输入框输入或语音结果写入时，上报 setTyping（单聊设对方 id，群聊设本方 id + groupId），防抖 2 秒后清除；发送时立即清除（见 [对方正在输入-功能说明.md](./对方正在输入-功能说明.md)）。

---

## 六、引用、反应、编辑、撤回

- **引用**：replyToMessage(msg) 设置 quoteTarget，发送时 metadata 带引用信息；MessageItem 内展示引用块。
- **反应**：addReaction(messageId, emoji) / removeReaction；metadata.reactions 为 { emoji: userId[] }。
- **编辑**：editMessage(messageId, content)；仅本人、UI 内联编辑。
- **撤回**：recallMessage(messageId)；仅本人，撤回后内容替换为「已撤回」等占位。

以上均在 chatSessionStore 内实现，无服务端交互。

---

## 七、消息免打扰

- **入口**：群信息面板（ChatGroupInfoPanel）内「消息免打扰」开关。
- **状态**：`mutedGroupIds: string[]`，toggleMuteGroup(groupId) 切换；侧边栏对免打扰群有视觉区分（如图标）。
- **范围**：前端 Mock，不涉及推送/声音；服务端方案见 [消息免打扰-服务端方案.md](./消息免打扰-服务端方案.md)。

---

## 八、交易卡片

- **类型**：MessageType.TRADE_CARD，metadata.tradeCard 为 TradeCardPayload（仿币安）。
- **发送**：sendTradeCard(payload)；从 TradeCardShareModal 一键生成示例后 onShare(payload)。
- **展示**：MessageItem 内对 TRADE_CARD 渲染 TradeCard 组件。
