# 对方正在输入 - 功能说明

本文说明「对方正在输入」在好友/群聊会话（`/chat`）中的展示与 Mock 上报逻辑：单聊显示对方昵称，群聊显示多人或「N 人正在输入」，以及输入框输入时如何触发与清除。

---

## 一、功能概述

- **单聊（C2C）**：当对方（好友）处于「正在输入」状态时，主区域底部显示「{对方昵称} 正在输入...」和点点动画。
- **群聊**：当有成员正在输入时，显示「A、B 正在输入...」或「N 人正在输入...」。
- **数据来源**：Store 中 `typingByUser[userId]`（单聊）、`typingByGroup[groupId]`（群聊，值为 userId[]）；Mock 下由**本方输入**时主动上报并防抖清除，模拟「对方/群内有人输入」的效果。

---

## 二、状态与动作（Store）

**位置**：`src/store/chatSessionStore.ts`

| 状态 | 类型 | 含义 |
|------|------|------|
| typingByUser | Record<string, boolean> | 单聊：userId → 是否正在输入 |
| typingByGroup | Record<string, string[]> | 群聊：groupId → 正在输入的 userId 列表 |

**动作**：`setTyping(userId: string, isTyping: boolean, groupId?: string)`

- 若传 `groupId`：更新 `typingByGroup[groupId]`，isTyping 为 true 时把 userId 加入数组，为 false 时从数组移除。
- 若不传 `groupId`：更新 `typingByUser[userId] = isTyping`。

---

## 三、展示逻辑（ChatSessionMain）

- **typingUserIds**：
  - 单聊：`typingByUser[activeConversation.id]` 为 true 时取 `[activeConversation.id]`，否则 `[]`。
  - 群聊：`typingByGroup[activeConversation.id] ?? []`。
- **文案（typingLabel）**：
  - 单聊：`activeConversation.name + " 正在输入"`（对方昵称）。
  - 群聊 1 人：`friends.find(id)?.name + " 正在输入"`。
  - 群聊多人：`names.join('、') + " 正在输入"` 或 `"N 人正在输入"`。
- 当 `typingUserIds.length > 0` 时，在消息列表底部渲染 `.chat-session-typing`：点点动画 + `{typingLabel}...`。

---

## 四、Mock 上报逻辑（ChatSessionInput）

- **输入时**：textarea `onChange` 或语音 `onResult` 时调用 `reportTyping(true)` 并 `scheduleTypingStop()`。
  - 单聊：`setTyping(activeConversation.id, true)`（对方 id）。
  - 群聊：`setTyping(CURRENT_USER_ID, true, activeConversation.id)`（本方在群内输入）。
- **防抖清除**：`scheduleTypingStop` 内设置 2 秒定时器，到期后 `reportTyping(false)`；每次新输入会重置该定时器。
- **发送时**：`handleSend` 内先清除定时器再 `reportTyping(false)`，然后发送消息。
- **切换会话**：draftId 变化时 effect 清理函数清除定时器，避免在错误会话上执行清除。

---

## 五、相关文件

| 类型 | 路径 |
|------|------|
| Store | `src/store/chatSessionStore.ts`（typingByUser、typingByGroup、setTyping） |
| 主区域 | `src/components/chat/ChatSessionMain.tsx`（typingUserIds、typingLabel、.chat-session-typing） |
| 输入区 | `src/components/chat/ChatSessionInput.tsx`（reportTyping、scheduleTypingStop、onChange/onResult 调用） |
| 样式 | `src/app/globals.css`（.chat-session-typing、.typing-dots） |

真实场景下应由对方设备或服务端在「开始输入/停止输入」时上报，本实现为前端 Mock 演示。
