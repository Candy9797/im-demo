# 交易卡片分享 - 功能说明

本文说明「仿币安」风格的交易卡片消息：数据结构、一键生成、分享到当前会话、列表展示等。仅用于好友/群聊会话（`/chat`），不涉及智能客服 IM。

---

## 一、功能概述

- **场景**：在好友或群聊中分享一条「交易成交通知」类卡片（如买入/卖出 BTCUSDT、价格、数量、盈亏等）。
- **入口**：`/chat` 输入区工具栏「分享交易卡片」按钮 → 打开弹窗，可一键生成示例或手动改数据后「分享到当前会话」。
- **展示**：消息列表中该条消息以卡片形式展示（深色主题、买卖色、盈亏色等），与文本/图片等并列。

---

## 二、数据结构

**TradeCardPayload**（`src/sdk/types.ts`）：

| 字段 | 类型 | 说明 |
|------|------|------|
| symbol | string | 交易对，如 BTCUSDT |
| side | "buy" \| "sell" | 买/卖 |
| price | string | 成交价 |
| quantity | string | 数量 |
| quoteAmount | string? | 成交额（可选） |
| pnl | string? | 盈亏金额（可选） |
| pnlPercent | string? | 盈亏比例（可选） |
| time | number | 成交时间戳 |
| fee | string? | 手续费（可选） |

消息类型为 `MessageType.TRADE_CARD`，`metadata.tradeCard` 存上述 payload；`content` 可为简短文案如「买入 BTCUSDT」。

---

## 三、发送流程

1. 用户点击工具栏「分享交易卡片」→ `TradeCardShareModal` 打开。
2. 弹窗内展示当前 payload 的预览（TradeCard 组件）、「一键生成示例」按钮、「分享到当前会话」按钮。
3. 「一键生成示例」：用随机或固定示例更新 payload（DEMO_PAYLOAD 可带随机数）。
4. 「分享到当前会话」：调用 `onShare(payload)` → ChatSessionInput 传入的 `sendTradeCard(payload)`。
5. **sendTradeCard**（chatSessionStore）：根据当前 `activeConversation` 确定 conversationKey，向 `messagesByConv[key]` 追加一条 `type: TRADE_CARD`、`metadata.tradeCard: payload` 的消息，sender 为当前用户。

---

## 四、展示（MessageItem）

- 在 `MessageItem` 的 `renderContent()` 中，对 `MessageType.TRADE_CARD` 分支：若存在 `message.metadata?.tradeCard`，渲染 `<TradeCard payload={...} />`，否则占位文案「[交易卡片]」。
- **TradeCard** 组件：根据 side、pnl 等渲染买卖颜色、盈亏颜色、时间格式化等，样式类如 `.trade-card`、`.trade-card-side.buy/sell`、`.trade-card-pnl.profit/loss`。
