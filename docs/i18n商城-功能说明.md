# i18n 商城 - 功能说明

本文说明带国际化的商城模块：路由与 locale、中间件重定向、多语言文案、列表/详情/购物车/结算、语言切换等。

---

## 一、功能概述

- **入口**：访问 `/shop` 或 `/shop/xxx` 时，由中间件重定向到 `/[locale]/shop` 或 `/[locale]/shop/xxx`，其中 locale 为 `zh` 或 `en`。
- **locale 来源**：优先 Cookie `NEXT_LOCALE`，其次请求头 `Accept-Language`，默认 `zh`。
- **路由**：`/[locale]/shop`（列表）、`/[locale]/shop/[id]`（详情）、`/[locale]/shop/cart`（购物车）、`/[locale]/shop/checkout`（结算）。
- **文案**：`messages/zh.json`、`messages/en.json`，通过 `IntlProvider` 注入，组件内使用 `useTranslations()` 取 key。

---

## 二、中间件（middleware.ts）

- **匹配**：`/shop`、`/shop/:path*`、`/zh/shop`、`/zh/shop/:path*`、`/en/shop`、`/en/shop/:path*`。
- **逻辑**：
  - 若已是 `/[locale]/shop` 形式且 locale 合法，直接 next()。
  - 若是 `/shop` 或 `/shop/xxx`，则取 locale（Cookie 或 Accept-Language），重定向到 `/${locale}/shop${rest}`，并设置 Cookie `NEXT_LOCALE`。
- **目的**：统一在 URL 中带 locale，便于服务端与客户端共用同一套路由和文案。

---

## 三、布局与文案注入（[locale]/layout.tsx）

- **Layout**：`[locale]` 下所有子路由共用该 layout；params.locale 经 `getLocale()` 校验后得到合法 locale。
- **getMessages(locale)**：异步加载 `messages/${locale}.json`（或通过 translations 模块）。
- **IntlProvider**：包裹子节点，传入 `locale`、`messages`；key 使用 locale 以保证切换语言时重新挂载。
- **Suspense**：params 与 getMessages 为异步，用 Suspense + fallback 避免 blocking-route。

---

## 四、商城页面与能力

| 路径 | 页面 | 说明 |
|------|------|------|
| /[locale]/shop | 列表页 | 商品列表、筛选、搜索、分页；使用 useTranslations 取「标题」「加入购物车」等 |
| /[locale]/shop/[id] | 详情页 | 单商品信息、加入购物车 |
| /[locale]/shop/cart | 购物车 | 购物车列表、数量修改、去结算 |
| /[locale]/shop/checkout | 结算 | 订单确认、提交（可为 Mock） |

购物车状态通常由全局 store（如 cartStore）管理，与 locale 解耦；仅展示文案随 locale 变化。

---

## 五、语言切换

- **组件**：如 ShopLocaleSwitcher、ShopHeader 内语言切换；调用 `switchLocalePath(currentPathname, targetLocale)` 得到目标 URL，再 `router.push` 或 Link 跳转。
- **switchLocalePath**：将当前 pathname 中的第一段 locale 替换为 targetLocale，保持其余路径与 query 不变。
- **Cookie**：中间件在重定向时已写入 `NEXT_LOCALE`，下次访问 `/shop` 会优先用该 locale。

---

## 六、shop2 与瀑布流

- **路由**：`/shop2`，非 `[locale]` 段，可为 SSR 瀑布流演示；若需 i18n 可再包一层 locale 或复用现有 IntlProvider。

---

## 七、相关文件

| 类型 | 路径 |
|------|------|
| 中间件 | `src/middleware.ts` |
| i18n 配置 | `src/lib/i18n.ts`（locales、defaultLocale、getLocale、shopPath、switchLocalePath） |
| 文案加载 | `src/lib/translations.ts`、`messages/zh.json`、`messages/en.json` |
| Layout | `src/app/[locale]/layout.tsx` |
| Provider | `src/components/providers/IntlProvider.tsx` |
| 页面 | `src/app/[locale]/shop/page.tsx`、`[id]/page.tsx`、`cart/page.tsx`、`checkout/page.tsx` |
| 组件 | `src/components/shop/ShopHeader.tsx`、`ShopLocaleSwitcher.tsx`、`ShopFilters.tsx`、`ShopProductFeed.tsx` 等 |
| Hook | `src/hooks/useLocale.ts`（若存在） |

脚本 `pnpm run i18n:translate` 可用于文案翻译或同步（若在 package.json 中配置）。
